# Communication Ltd

אפליקציית Django לחברת תקשורת — ניהול משתמשים ואפליקציה.

## טכנולוגיות
- **Python** · **Django**

## הרצה
```bash
pip install -r requirements.txt  # או venv
python manage.py migrate
python manage.py runserver
```

## מבנה
```
communication_ltd/    # פרויקט Django
users/                # אפליקציית משתמשים
config.py
manage.py
```

## הערה
- `db.sqlite3` — לא נכלל ב-git (בדוק .gitignore)
- הרצי `migrate` לפני הפעלה ראשונה
